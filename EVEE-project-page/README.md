# EVEE Project Page

Static project page for EVEE.

## Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## GitHub Pages

Publish this repository from:

- Branch: `main`
- Folder: `/root`

The expected GitHub Pages URL is:

```text
https://zejzhao.github.io/EVEE-project/
```

The EVEE code repository remains separate:

```text
https://github.com/ZeJZhao/EVEE
```

Only the static website assets are stored here.
